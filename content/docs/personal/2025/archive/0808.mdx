## 풀어본 문제들

### 1. 정수 찾기

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181840)
- **핵심**: 배열에서 특정 정수가 포함되어 있는지 확인
- **학습 포인트**: includes() 메서드를 활용한 배열 내 요소 존재 여부 확인

### 2. 배열의 원소 삭제하기

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181844)
- **핵심**: 특정 조건을 만족하는 원소들을 배열에서 제거
- **학습 포인트**: filter() 메서드를 사용한 배열 필터링

### 3. 꼬리 문자열

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181841)
- **핵심**: 조건에 맞는 문자열들을 연결하여 새로운 문자열 생성
- **학습 포인트**: 조건문과 문자열 연결 연산자 활용

### 4. 0 떼기

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181847)
- **핵심**: 문자열 앞쪽의 불필요한 0을 제거
- **학습 포인트**: 정규표현식 또는 parseInt() 메서드를 활용한 0 제거

### 5. 부분 문자열

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181842)
- **핵심**: 한 문자열이 다른 문자열에 포함되는지 확인
- **학습 포인트**: includes() 메서드를 사용한 부분 문자열 검사
