## 알고리즘 문제 추가 프로세스

### 실행 방법

`yarn add-algorithm` 명령어를 실행하여 대화형으로 알고리즘 문제를 추가합니다.

### 기능

1. **/자동 날짜 처리**: 오늘 날짜가 `src/data/algorithm-records.json`에 없으면 새로운 날짜 필드를 자동으로 생성합니다.

2. **대화형 입력**: 다음 정보를 순차적으로 입력받습니다:

   - 문제 제목
   - 플랫폼 (예: 프로그래머스, 백준, 리트코드)
   - 문제 링크
   - 태그 (쉼표로 구분하여 입력)

3. **자동 저장**: 입력된 정보를 JSON 파일에 자동으로 저장합니다.

### 사용 예시

```bash
yarn algorithm
```

스크립트가 실행되면 다음과 같이 입력을 요청합니다:

```
오늘 날짜 (2025.07.31)에 문제를 추가합니다.

문제 제목: 두 수의 합
플랫폼: 프로그래머스
링크: https://school.programmers.co.kr/learn/courses/30/lessons/1
태그 (쉼표로 구분): 해시테이블, 배열

✅ 문제가 성공적으로 추가되었습니다!
```
